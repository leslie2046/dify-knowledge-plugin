# Privacy Policy

This plugin sends the user's retrieval query and optional retrieval parameters to the Dify Knowledge API endpoint configured in the plugin credentials.

The plugin does not persist user queries, retrieval results, API keys, or dataset identifiers outside the Dify runtime. Network traffic is sent only to the configured Dify API base URL.

You are responsible for providing a valid Dify Service API key and for ensuring that the configured endpoint complies with your organization's privacy and data handling requirements.
